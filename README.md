# bypass-api
Bypass API is an API that allows you to skip URL shorteners.

## Supported URLs
- all general redirects like: bit.ly, tinyurl.com, etc.
- boost.ink
- linkvertise.net, link-to.net, direct-link.net, up-to-down.net (all linkvertise links)
- shortly.xyz
- sub2unlock.com
- shortconnect.com
